# 🌌 Lore

Inside the Base network, fragments of digital energy slowly began forming living creatures called **BaseBlobs**.

These small beings absorb energy from transactions, blocks, and network activity.

Some are sleepy.
Some are chaotic.
Some resonate with rare cosmic signals.

Nobody fully understands where they came from.
But explorers discovered one thing:

> *The deeper a Blob travels into the network… the stranger it becomes.*

---

# 🎯 Core Gameplay Loop

```
Expeditions → Cubes → Upgrades → Evolution → New Zones
```

Players:
- send Blobs on expeditions across 6 unique zones
- collect 💠 Cubes
- level up Blobs and upgrade their abilities
- evolve Blobs into stronger forms
- summon new Blobs and expand their collection

---

# 👾 Personality System

Each Blob has a unique personality that shapes its stats and playstyle:

| | Personality | Bonus |
|--|-------------|-------|
| 😊 | **Happy** | +20% XP from all expeditions |
| 😴 | **Sleepy** | Energy regenerates 30% faster |
| 🍀 | **Lucky** | +15% cube reward on all zones |
| ⚡ | **Chaotic** | 30% chance for ×1.5 cube jackpot |
| 🌌 | **Cosmic** | +25% all rewards at Lv.10+ |

# ⚡ Energy System

Energy powers expeditions and regenerates over time.

|------|-------|

# ⬆️ Upgrade System

Each Blob has **3 upgrade branches**, each with 5 levels. Slots unlock through evolution.

| Branch | Effect | Max Bonus |
|--------|--------|-----------|
| ⚡ Speed | Reduces expedition time | −45% |
| 💰 Harvest | More cubes per expedition | +75% |
| 🍀 Fortune | Bonus event chance | +45% |

**Upgrade slot progression:**
- **Lv. 1** → 1 branch available
- **Lv. 5** → 2 branches available
- **Lv. 10** → All 3 branches available

# 📈 Progression & Evolution

## Max Level
20

## Evolution Stages

| Level | Form | What Changes |
|-------|------|-------------|
| Lv. 1–4 | 🟦 Base Blob | Standard form |
| Lv. 5 | ✨ Glow Form | Neon aura ring, 2nd upgrade branch unlocked |
| Lv. 10 | 💎 Crystal Form | Orbiting particles, all upgrades unlocked |
| Lv. 20 | 👑 Ascended Form | Golden glow, unique personality passive |

## Ascended Passives (Lv. 20)

| Personality | Passive |
|-------------|---------|
| 😊 Happy | +50% XP from all sources |
| 😴 Sleepy | Base Fields costs 0 energy |
| 🍀 Lucky | Guaranteed double drop once per day |
| ⚡ Chaotic | Jackpot multiplier ×3 (was ×1.5) |
| 🌌 Cosmic | +100% rewards in Base Chain Core |

# 🐣 Blob Collection

## Summoning
- **Cost:** 1500 💠 Cubes
- Receive a random personality Blob starting at Lv. 1

## Gallery
Browse your full Blob collection, view stats, and choose which Blob to send on expeditions.

---

# 📋 Quest System

## Daily Quests
Reset every 24 hours:

| Quest | Reward |
|-------|--------|
| Complete 1 expedition | 💠 40 + 50 XP |
| Collect 80 Cubes | 💠 40 + 35 XP |
| Send 2 expeditions | 💠 25 + 30 XP |

## Main Quests
Long-term progression goals — reach levels, complete expeditions, summon Blobs, unlock zones.
